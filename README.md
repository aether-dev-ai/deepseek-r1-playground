# DeepSeek R1 Playground 🧠

一个可视化 DeepSeek R1 推理过程的交互式 Web 应用。

![DeepSeek R1 Playground](https://img.shields.io/badge/Next.js-16-black?logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5.7-blue?logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.4-38B2AC?logo=tailwind-css)

## ✨ 特性

- 🔍 **实时可视化**：逐步展示 AI 的推理过程
- 🎨 **美观界面**：基于 Tailwind CSS 的现代化 UI
- 🚀 **快速部署**：一键部署到 Vercel
- 💡 **教育价值**：理解 AI 如何思考和推理
- 🌓 **暗黑模式**：自动适配系统主题

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 本地运行

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看效果。

### 构建部署

```bash
npm run build
npm start
```

## 📦 部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/aether-dev-ai/deepseek-r1-playground)

**🚀 Live Demo**: [https://deepseek-r1-playground.vercel.app](https://deepseek-r1-playground.vercel.app)

1. 点击上方按钮
2. 连接你的 GitHub 仓库
3. 自动部署完成！

## 🛠️ 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **部署**: Vercel

## 💡 使用说明

1. 在输入框中输入你的问题
2. 点击"开始推理"按钮
3. 观察 AI 的思维过程：
   - 🤔 **思考步骤**：分析问题结构
   - ⚡ **行动步骤**：检索和处理信息
   - 💡 **结论步骤**：综合得出答案
4. 查看最终的完整回答
